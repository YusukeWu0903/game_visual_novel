# 視覺小說開發專案 - 技術規格書

## 核心設計理念
本專案採用「數據與表現分離」的現代 Web 遊戲開發架構，利用 AI 自動化管線極大化生產效率。

## 技術棧
- **框架**: Vite + 自研 JS 控制層
- **自動化管線**:
    - `optimize_assets.py`: 自動將原始圖片處理為 1080p WebP，優化載入效能。
    - `generate_script.py`: 將結構化 JSON 劇本轉譯為遊戲引擎指令，實現自動化量產。

## 面試亮點 (Why this project?)
1. **工程化生產力**: 開發者無需手動調整圖片尺寸，亦無需手動寫複雜的引擎 code，專注於劇本敘事。
2. **可擴充性**: 支援多角色立繪動態定位與並排，架構已預留過場特效與事件觸發接口。
3. **性能優化**: 透過 WebP 與動態載入機制，確保在瀏覽器上達到流暢遊玩體驗。
